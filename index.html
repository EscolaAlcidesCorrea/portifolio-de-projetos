Biblioteca EE. Prof Alcides Correa: Uma Webpage Personalizada

O projeto é uma página web criada com HTML e CSS, destinada a exibir algumas idicações de livros favoritos dos alunos da escola. O objetivo é criar um ambiente virtual onde seja possível compartilhar seus livros preferidos, fornecendo uma descrição breve de cada um, incluindo o autor e o ano de publicação.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

Cada livro é destacado em seu próprio contêiner, mostrando uma imagem da capa, detalhes do livro e um link de compra. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

Em Busca da Cidade Perdida

Narrativa interativa: A história se desenvolve em parágrafos, e a cada momento crucial, o jogador deve escolher como prosseguir.

Múltiplos finais: As escolhas ao longo da jornada levam a diferentes desfechos — é possível encontrar a cidade perdida, abandonar a missão ou chegar a outros destinos inesperados.

Navegação simples: A experiência é guiada por botões, que permitem avançar na narrativa de forma intuitiva.

Como jogar: Abra o arquivo index.html em seu navegador. Leia atentamente o trecho da história apresentado. Escolha uma das opções disponíveis clicando no botão correspondente. Continue tomando decisões até alcançar um dos finais possíveis.

O jogo é composto por três arquivos principais:

index.html: Estrutura a página, exibindo a narrativa e os botões de interação.

style.css: Define o visual do jogo — cores, fontes e layout responsivo.

script.js: Controla a lógica do jogo, gerenciando as escolhas e o progresso da história.
